用R语言告白吧！

2020-06-27

高逼格表白——用R语言绘制小心心

1.加载ggplot包,设定数据集

2.用geom_line画出大致轮廓，并设置边框颜色； 用geom_polygon来进行颜色填充并隐藏图例; scale_x/y_continuous隐藏坐标轴

3.theme_bw设置背景色，panel.grid删除网格线、panel.border删除边框线、axis.ticks删除刻度线、axis.title删除坐标轴标签

4.在心上写上你想说的话：label

代码如下

library(ggplot2)

t = seq(0, 2 * pi, by = 0.1)
x = 16 * sin(t)^3
y = 13 * cos(t) - 5 * cos(2 * t) - 2 * cos(3 * t) - cos(4 * t)
a = (x - min(x))/(max(x) - min(x))
b = (y - min(y))/(max(y) - min(y))

ggplot(data=NULL, aes(x=x, y=y)) +
geom_line(aes(color=I('white'))) +
geom_polygon(aes(fill='red'), show.legend = F) +
scale_x_continuous(labels = NULL) +
scale_y_continuous(labels = NULL) +

theme_bw() + 
theme(panel.grid.major = element_blank(), 
panel.grid.minor = element_blank(), 
panel.border = element_blank(), 
axis.ticks = element_blank(), 
axis.title = element_blank()) + 
 annotate('text', x=median(a), y=median(b),
          
label='Wish Scuer happy everyday!
  ——yxzoe',
 size=5)
